# Gmail Treemax

Visualize your unread Gmail emails by sender using an interactive treemap.

## Quick Start

1. Audit the repo
2. Run ```npx github:pierrecorsini/gmail_treemax```

This will:
1. Start a local development server
2. Open your browser at `http://localhost:3000`
3. Follow the instructions to get your OAuth Client ID, connect your Gmail account, and see your email treemap !
